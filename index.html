How to Use the Outdoor Data Explorer

The Outdoor Data Explorer provides advanced visualization tools for analyzing particle monitoring data from the ROCIS project. Follow these steps to explore your data:

  1. Select a Cohort: Use the dropdown menu to choose from the available cohorts.
  2. Choose Particle Type: Select between "Total Particles" or "Large Particles" using the dropdown to focus on the specific particle measurements you want to analyze.
  3. Apply Pennsylvania Filter: Check the "Pennsylvania Only" option to filter data specifically for participants outside Pennsylvania.
  4. Load Your Data: Click the "Load Dataset" button to generate the interactive visualization with your selected parameters.
  5. Explore the Interactive Chart:
    • Hover for Details: Move your mouse over data points to see specific values and timestamps
    • Toggle Scale: Switch between linear and logarithmic scales using the buttons below the chart
    • Zoom and Pan: Click and drag to zoom into specific areas of the chart. Use mouse wheel to zoom in/out. Hold shift and drag to pan across different time periods. Double-click on the white space inside the graph to reset to original view
    • Legend Control: Click on legend items to show/hide specific data series. This allows you to focus on particular sensors or data streams
    • Data Sampling: Choose between "All points", "Every 5th", or "Every 10th" to control how many data points are displayed. Large datasets with all points may load slowly, so using "Every 5th" or "Every 10th" speeds up visualization while maintaining the overall pattern. This only affects display speed - your complete data is always preserved and available for download.
  6. Compare Data (Optional): Use the "Compare Large vs Total" mode to view side-by-side charts comparing both particle types for the same cohort.

Tips for Best Results:

  • Start with a specific cohort to focus your analysis
  • Use the Pennsylvania filter if you're studying participants outside Pennsylvania
  • Try both linear and log scales to better understand data patterns
  • The comparison mode is particularly useful for understanding differences between particle types
  • Use data sampling ("Every 5th" or "Every 10th") when working with large datasets for faster performance
